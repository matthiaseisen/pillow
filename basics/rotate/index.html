Rotating Images

1   Preparations

Import Pillow:

from PIL import Image

Load the original image:

img = Image.open("flowers.jpg")

The original image

2   Counterclockwise Rotation

img2 = img.rotate(45)

Rotated by 45 degrees

img3 = img.rotate(90)

Rotated by 90 degrees

3   Clockwise Rotation

img4 = img.rotate(-45)

Rotated by 45 degrees (clockwise)

4   Disable cropping of the output image

img5 = img.rotate(45, expand=True)

Uncropped version of rotated image

5   Apply a resampling filter

img6 = img.rotate(45, resample=Image.NEAREST)

Use nearest neighbor (default)

img7 = img.rotate(45, resample=Image.BILINEAR)

Use linear interpolation

img8 = img.rotate(45, resample=Image.BICUBIC)

Use cubic spline interpolation
